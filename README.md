# Rova Solution

نظام إدارة متكامل مبني على .NET 9.0

## المشاريع

- **Rova.Core**: المنطق الأساسي والخدمات
- **Rova.Shared**: النماذج والأنواع المشتركة
- **Rova.Web**: تطبيق الويب (قيد التطوير)
- **Rova.Mobile**: تطبيق الموبايل (قيد التطوير)

## البناء

```bash
dotnet build
```

## المتطلبات

- .NET 9.0 SDK
